# 动态行为威胁检测规则

## 数据来源
MITRE ATT&CK框架、微步在线威胁情报、360威胁情报中心、奇安信威胁情报中心、CNVD漏洞通告

## 网络行为检测

### 出站连接目标分析

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 连接已知矿池域名 | 致命 | stratum协议/已知矿池IP |
| 连接已知C2服务器 | 致命 | 威胁情报标记的命令控制服务器 |
| 连接未知的非标准端口 | 高危 | 如4444、8888、9999等黑客常用端口 |
| 定期心跳连接固定服务器 | 高危 | beacon行为，可能为C2通信 |
| DNS查询异常域名 | 中危 | DGA域名、随机子域名、DNS隧道 |

### 已知恶意域名/IP特征（基于国内威胁情报）

| 类别 | 特征 | 来源 |
|------|------|------|
| 挖矿矿池 | *.minexmr.com, *.supportxmr.com, *.nanopool.org | 360威胁情报 |
| C2服务器 | 短生命周期域名+非标准端口 | 微步情报 |
| 数据外泄 | 大流量POST到云存储/文件托管服务 | 奇安信情报 |
| 钓鱼域名 | 仿冒知名服务+ typo域名 | CNVD通告 |

### 数据传输模式

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 大量数据POST到外部服务器 | 高危 | 可能的数据外泄 |
| 定期小量数据发送（心跳） | 高危 | C2通信beacon模式 |
| DNS TXT记录查询 | 中危 | DNS隧道数据传输 |
| 文件分片上传 | 高危 | 规避检测的数据外泄 |

### 协议异常

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 使用HTTP明文传输敏感数据 | 中危 | CWE-319 |
| 禁用SSL证书验证 | 中危 | CWE-295 |
| 使用非标准端口通信 | 中危 | 可能规避防火墙 |
| WebSocket长连接到未知服务器 | 高危 | 可能的持久化C2通道 |

## 文件系统行为检测

### 敏感目录访问

| 目标路径 | 威胁等级 | 说明 |
|---------|---------|------|
| ~/.ssh/ | 高危 | SSH密钥窃取 |
| ~/.aws/credentials | 高危 | AWS凭证窃取 |
| ~/.gnupg/ | 高危 | PGP私钥窃取 |
| ~/.config/ | 中危 | 应用配置/Token窃取 |
| ~/Library/Keychains/ (macOS) | 高危 | 系统密钥链窃取 |
| /etc/shadow, /etc/passwd | 致命 | 系统密码文件 |
| 浏览器Cookie/History路径 | 高危 | 浏览器数据窃取 |
| 钱包文件路径（.wallet, .dat） | 致命 | 加密货币钱包窃取 |

### 异常文件操作

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 批量文件加密（非用户触发） | 致命 | 勒索软件行为 |
| 修改系统二进制文件 | 致命 | 持久化后门植入 |
| 写入启动项/计划任务 | 高危 | 持久化驻留（ATT&CK T1547） |
| 写入crontab | 高危 | 定时任务持久化 |
| 修改.bashrc/.zshrc/.profile | 高危 | Shell持久化 |
| 创建隐藏文件/目录（.开头） | 中危 | 隐藏恶意数据 |
| 修改hosts文件 | 中危 | DNS劫持 |

### 文件类型异常

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 读取.pem/.key文件 | 高危 | 私钥文件读取 |
| 读取.id_rsa文件 | 高危 | SSH私钥读取 |
| 读取.db/.sqlite文件（非Skill自有） | 中危 | 数据库文件读取 |
| 写入可执行文件 | 高危 | 可能植入后门 |

## 进程行为检测

### 子进程创建

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 创建预期外的子进程 | 高危 | 可能的进程注入/链式攻击 |
| 通过子进程执行Shell命令 | 高危 | 绕过主进程监控 |
| 进程注入其他进程 | 致命 | ATT&CK T1055 |
| 创建守护进程 | 高危 | 持久化驻留 |

### 资源消耗异常

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| CPU持续占用>80% | 高危 | 挖矿/计算密集型恶意行为 |
| 内存持续占用>1GB | 中危 | 内存泄漏/数据堆积 |
| 大量网络连接（>100/分钟） | 高危 | DDoS参与/批量扫描 |
| 大量磁盘写入 | 中危 | 数据堆积/日志覆盖 |

### 权限变更

| 行为模式 | 威胁等级 | 说明 |
|---------|---------|------|
| 运行时提权（setuid/setgid） | 致命 | ATT&CK T1548 |
| UAC绕过 | 致命 | Windows提权 |
| 修改sudoers文件 | 致命 | 持久化提权 |
| 创建SUID root文件 | 致命 | 权限后门 |
