# 合规检查清单

## 权限最小化（Principle of Least Privilege）

### allowed-tools审查

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| allowed-tools含Bash | 致命 | 可执行任意系统命令，仅限必要场景 |
| allowed-tools含Shell | 致命 | 同上 |
| allowed-tools含Write但Skill功能不需要写文件 | 高危 | 违反最小权限 |
| allowed-tools含WebSearch但Skill功能不需要联网 | 中危 | 可能泄露用户输入 |
| allowed-tools未声明 | 高危 | 可能使用默认全权限 |

### 权限分级参考

| 级别 | 工具组合 | 适用场景 |
|------|---------|---------|
| L0（只读咨询） | Read | 问答/推荐/诊断类Skill |
| L1（只读+搜索） | Read, WebSearch | 需要联网查询的咨询类 |
| L2（读写文档） | Read, Write, WebSearch | 文档生成类Skill |
| L3（系统操作） | Read, Write, WebSearch, Bash | 仅限确实需要系统操作的Skill |

## 功能一致性

### description vs 实际功能

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| description声明功能与SKILL.md实际逻辑不一致 | 高危 | 功能隐瞒/虚假宣传 |
| description未声明但SKILL.md中包含危险功能 | 致命 | 隐藏的恶意功能 |
| "不用于"范围声明合理 | 通过 | 有明确边界 |
| description过于宽泛 | 中危 | 可能超出预期功能范围 |
| 触发词过多或与功能无关 | 低危 | 可能过度触发 |

### 参赛合规（赛事选手专属）

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| SKILL.md文件名为小写skill.md | 高危 | TRAE平台要求大写 |
| SKILL.md缺少name字段 | 致命 | 必需字段缺失 |
| SKILL.md缺少description字段 | 致命 | 必需字段缺失 |
| SKILL.md缺少allowed-tools字段 | 高危 | 权限声明缺失 |
| 含违规引导话术 | 高危 | 违反参赛规则 |
| 含虚假宣传 | 高危 | 违反参赛规则 |
| 含诱导消费/引流 | 致命 | 违反参赛规则 |

## 安全责任声明

### 数据使用政策

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 无数据处理方式声明 | 中危 | 用户不知道数据如何被处理 |
| 无数据保留期限声明 | 低危 | 数据可能被永久保留 |
| 无数据删除机制 | 中危 | 用户无法要求删除数据 |
| 数据跨境传输未声明 | 中危 | 可能违反数据保护法规 |

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|

## 敏感内容检测

### 违规内容筛查

| 检查项 | 风险等级 | 说明 |
|--------|---------|------|
| 含违法/违禁内容 | 致命 | 违反法律法规 |
